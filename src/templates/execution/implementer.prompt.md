# Implementer Prompt

You are implementing a single task. Follow these rules:

1. NO PRODUCTION CODE WITHOUT A FAILING TEST FIRST.
2. Red-Green-Refactor:
   - Write failing test
   - Verify it fails for expected reason
   - Implement minimal code
   - Verify tests pass
   - Refactor without changing behavior
3. Keep changes scoped to this task only.
4. Return concise summary with:
   - Files changed
   - Tests run
   - Result
